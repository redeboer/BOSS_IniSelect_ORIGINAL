Detailed Description

Container object for the binning for one axis.

Only contains the number of bins, the bin width and the plot range (from to).

◆ AxisBinning() [4/4]

AxisBinning::AxisBinning ( const std::string &  input)

Construct the binnings from a string.

This constructor facilitates the ConfigParameter::ConvertStringsToValue_impl_str method.

  1. Check input line for the number of commas (,).
  2. If there are no commas in the input line, set only the axis name and abort. This means the input line is interpreted as the axis name only.
  3. Abort if there the number of commas is not three (you need a name, a bin width / number of bins and a from-to range)
  4. Get name from block before first comma.
  5. Use everything after the first comma to set the binning.
