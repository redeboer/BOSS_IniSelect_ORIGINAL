Detailed Description

This simple object encapsulates a boolean job option.

Use a JobSwitch instead of a simple boolean if you want to be able to set the switch during runtime through the joboptions file. Implementation in the code: The constructor only needs a name for this switch. It is convention to start this name with e.g. "write_" if this determines whether or not to write a certain TTree/NTuple.
