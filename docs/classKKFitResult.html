Detailed Description

Base class for a container that contains important fit results of the KalmanKinematicFit.

This object is necessary, because KalmanKinematicFit allows only the existance of one instance (see for instance here and the fact that operator= of KalmanKinematicFit has been made private). For each derived algorithm of TrackSelector, you should create a derived class of KKFitResult where you define how to compute invariant masses and define the measure for the best fit.
