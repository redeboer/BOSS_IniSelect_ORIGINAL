Detailed Description

WIP Analyse events.

Based on the original RhopiAlg package, but now making use of the TrackSelector base algorithm and its NTuple booking procedure.

Author
Remco de Boer (r.e.d.nosp@m.eboe.nosp@m.r@stu.nosp@m.dent.nosp@m.s.uu..nosp@m.nl or remco.nosp@m..de..nosp@m.boer@.nosp@m.ihep.nosp@m..ac.c.nosp@m.n)
Date
December 27th, 2018
