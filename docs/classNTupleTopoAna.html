◆ index

NTuple::Item<int> NTupleTopoAna::index

Number of MC particles stored for this event. This one is necessary for loading following two items, because they are arrays.

Remarks
It is called index, because you use it as an index in the WriteMcTruthForTopoAna loop when filling the particle and mother arrays. Of course, when you exit the loop, this index also gives the number of items.
