void D0omega_K4pi::FindBestKinematicFit ( )
privatevirtual

Perform a Kalman 4-constraint Kalman kinematic fit for all combinations and decide the combinations that results in the 'best' result.

The 'best' result is defined as the combination that has the smallest value of: (that is the combination for which the invariant mass of the is closest to ). See D0omega_K4pi::MeasureForBestFit for the definition of this measure.
