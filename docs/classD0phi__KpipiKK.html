Detailed Description

Analyse events.

This package has been created as an extension of the D0phi_KpiKK package. The motivation for reconstructing the meson through the decay channel (extra ) is that (1) its branching ratio is much larger ( versus for ) and (2) that the extra excludes the that we see as a large backround in the D0phi_KpiKK package (due to parity). See PDG listing.
