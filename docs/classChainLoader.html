Detailed Description

A container class for a TChain that allows easy access to its branches.

Addresses are set automatically upon construction.

◆ BookAddresses()

void ChainLoader::BookAddresses ( bool  print_branches = false,
bool  print_averages = false 
)

Automatically book addresses for all branches of the underlying TChain.

The branches are accessible by name (use method Get_<type>, though you have to the type to use this).

Parameters
print_branchesWhether or not to print the TChain names and its branches+types. Use true for debugging purposes in particular.
print_averagesWhether or not to compute and print the averages of each branch. Use true for debugging purposes in particular. Using this setting costs time!
  1. Get list of TBranches.
  2. Loop over list of TBranches and determine typename of the branch. The data type of a branch can be determined from the last character of its title. See TTree for the labels of each type.

If branch title contains a [, this means it is an array and it will not be loaded.
